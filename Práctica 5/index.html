Etiquetas de estilo en HTML

Este es un texto de ejemplo para prácticar con las etiquetas de estilo.

Estilo negritas

La etiqueta b se utiliza para poner el texto en negritas. Por ejemplo: este texto está en negritas.

La etiqueta strong también se utiliza para poner texto en negritas tienen un significado semántico más fuerte.

Diferencias entre B y strong

Ambas etiquetas hacen que el texto se vea en negritas pero los motores de búsqueda y los lectores de pantalla pueden utilizar está información para entender mejor el contenido de la página.

Estilo Cursiva

La etiqueta i se utiliza para reescribir el texto en cursiva. Por ejemplo: este texto está en cursiva

La etiqueta em también se utiliza para poner el texto en cursiva, pero tiene un significado semántico de énfasis

Diferencias entre i y em

Ambas etiquetas hacen que el texto se vea impulsiva pero em al tene mayor énfasis semántico el lector de pantalla puede enfatizar la lectura de estás palabras.

Estilo subrayado

La etiqueta u se usan para subrayar el texto. Por ejemplo: este texto esta subrayado
